# Figure and Web UI Specification

## Scientific figures

1. representative spectra
2. local residual examples
3. scale by region
4. class comparison
5. SNR dependence
6. null test

Each final figure must be generated by `scripts/make_figures.py`, saved as SVG and 300 dpi PNG, include units and sample size, and have a sidecar JSON containing the Git commit and configuration hash.

## Web application

The `web-react/` folder is a Vite + React + Tailwind starter. Replace demo values with `results/summary.json` only after the scientific pipeline has run. Required sections:

- project purpose and novelty boundary;
- dataset/product manifest;
- key quantitative metrics with uncertainty;
- interactive or responsive figure gallery;
- methodology and validation tabs;
- assumptions and limitations;
- provenance/download panel;
- citation and licence.

Do not show fabricated live telemetry, fake APIs, fake detections or fake confidence values.
